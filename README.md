By maintaining this repository, I aim to create a strong habit and maintain an active streak on GitHub!

## 📁 Repository Structure

### `April month`
Currently focusing on mastering fundamental data manipulation and analysis skills.
*   **`01-Preprocessing-and-Cleaning/`**: Dealing with missing values, outliers, data imputation, encoding categorical variables, and feature scaling.
*   **`02-EDA-and-Visualization/`**: Exploring distributions, correlations, and building insightful charts using Matplotlib, Seaborn, or Plotly.
*   **`03-GroupBy-and-Aggregation/`**: Complex data aggregation, transformations, pivot tables, and multi-level indexing using Pandas.
*   **`04-SQL_Prep/`**: SQL queries, window functions, joins, subqueries, and database concepts.
*   **`datasets/`**: (Ignored by default if large) Datasets used for practice.
